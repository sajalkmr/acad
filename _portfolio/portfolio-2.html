Overview

STRbook is a comprehensive digital solution designed to replace traditional physical record books in educational institutions. This full-stack application streamlines the process of maintaining and accessing student transformation records.

Key Features

  • Secure Authentication: JWT-based authentication system for students and teachers
  • RESTful APIs: Well-designed CRUD operations for efficient data management
  • Optimized Database: PostgreSQL integration with performance-optimized queries
  • CI/CD Pipeline: Automated deployment using GitHub Actions

Technologies Used

  • Frontend: React, modern UI/UX design principles
  • Backend: Node.js, Express.js
  • Database: PostgreSQL
  • Security: JSON Web Tokens (JWT)
  • DevOps: GitHub Actions for CI/CD

Impact

Successfully deployed in college environment, STRbook has significantly reduced manual data entry time and improved accessibility of student records for both educators and students.
